Debug Gambar PNG

1. Verifikasi Gambar

Gambar berikut harus menampilkan overlay.png dengan latar belakang checkerboard:

2. Verifikasi Path

Jika gambar di atas tidak muncul, kemungkinan path file salah.

Path yang digunakan: images/overlay.png

Path lengkap seharusnya: /public/images/overlay.png

Checklist:

  • Pastikan file berada di lokasi yang benar
  • Periksa ekstensi file (.png)
  • Periksa nama file (perhatikan huruf besar/kecil)
  • Pastikan format PNG mendukung transparansi

Jika gambar muncul dengan benar di halaman ini tapi tidak di AR, maka masalahnya ada pada konfigurasi AR.js.
